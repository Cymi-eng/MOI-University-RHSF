# MOI UNIVERSITY MAIN ALTAR
## Overview
Welcome to Moi University Main Altar _ A ministry of repentance and Holiness Church website desgned to give a story about moi Univeristy main Altar Church and prepare saints for the Glorious coming of The Messiah.

This platform is designed to strengthen communication, share the Word of God, and connect members of the church community both on and off campus.

## Features
* Home Page – Church introduction and welcome message
* Gallery Section – church Gallery
* Weekly Program – Organized church schedule
* leadersip Page – Information on leadership structure
* Contact Form – Easy communication with church leaders
* Responsive Design – Works on mobile, tablet, and desktop
* Bible Verse Highlights – Inspirational scriptures throughout the site

## Built With
* HTML5
* Tailwind CSS (or CSS3)
* JavaScript
* GitHub Pages (Deployment)
